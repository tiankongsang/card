# Drawer 抽屉

有些时候, Dialog 组件并不满足我们的需求, 比如你的表单很长, 亦或是你需要临时展示一些文档, Drawer 拥有和 Dialog 几乎相同的 API, 在 UI 上带来不一样的体验。



# 代码演示

<template>
    <c-drawer v-model="flag" />
</template>

<script setup>

import { ref } from 'vue'

let flag = ref(false)

const changeState = () => flag.value = !flag.value;

</script>

# Props

属性说明类型可选值默认值
model-value是否显示 Drawerboolean——false
lock-scroll是否在 Drawer 出现时将 body 滚动锁定boolean——true
close-on-click-modal是否可以通过点击 modal 关闭 Drawerboolean——true
close-on-press-escape是否可以通过按下 ESC 关闭 Drawerboolean——true
close-delayDrawer 关闭的延时时间，单位毫秒number——0
modal是否需要遮罩层boolean——true
directionDrawer 打开的方向stringrtl / ltr / ttb / bttrtl
show-close是否显示关闭按钮boolean——true
sizeDrawer 窗体的大小string——30%
titleDrawer 的标题，也可通过具名 slot （见下表）传入string——
with-header控制是否显示 header 栏, 默认为 true, 当此项为 false 时, title attribute 和 title slot 均不生效boolean——true
